# 🎨 Duolingo 风格儿童积分系统设计指南

## 🌈 核心设计原则 (Core Design Principles)

### 视觉风格 (Visual Style)
- **大胆的色彩** - 使用高饱和度的主色调，营造活力氛围
- **超圆角设计** - 所有组件采用大圆角 (1.5rem+)，按钮使用pill形状
- **游戏化元素** - 丰富的emoji、动画效果和互动反馈
- **层次分明** - 清晰的信息架构和视觉层次

---

## 🎯 色彩系统 (Color System)

### 主色调
- **Duolingo 绿**: `#58CC02` - 主要按钮、成功状态
- **活力橙**: `#FF9600` - 奖励、警告、强调
- **天空蓝**: `#1CB0F6` - 信息、链接、次要操作

### 游戏化彩色板
- **紫色**: `#CE82FF` - 习惯打卡、特殊徽章
- **粉色**: `#FF9CE5` - 社交、排行榜
- **黄色**: `#FFC800` - 积分、成就、庆祝

### 中性色
- **温暖灰系**: `#F5F5F4` 到 `#1C1917` - 文本、背景、边框
- **使用场景**: 确保良好的对比度和可读性

---

## 📐 组件设计规范 (Component Guidelines)

### 按钮 (Buttons)
- **主要按钮**: pill形状 (`rounded-full`)，渐变背景，粗体文字
- **次要按钮**: 大圆角 (`rounded-[1.5rem]`)，描边样式
- **悬停效果**: 轻微放大 (`scale-105`) + 阴影增强
- **点击反馈**: 缩放动画 (`scale-95`)

### 卡片 (Cards)
- **圆角**: 统一使用 `rounded-[2rem]` (32px)
- **边框**: 4px粗边框，使用主题色
- **阴影**: 大阴影 (`shadow-2xl`) + 彩色阴影
- **背景**: 渐变背景增强视觉层次

### 导航 (Navigation)
- **标签页**: 垂直布局，图标+文字组合
- **激活状态**: 渐变背景 + 放大效果
- **装饰元素**: 彩色条纹、装饰圆点

---

## 🎮 交互模式 (Interaction Patterns)

### 动画效果
- **入场动画**: 弹簧动画 (`type: "spring"`)
- **悬停反馈**: 旋转、放大、颜色变化
- **点击反馈**: 立即的缩放反馈
- **庆祝动画**: 彩带、粒子效果、旋转

### 游戏化元素
- **连击系统**: 火焰图标 + 数字
- **进度条**: 圆形进度，彩色填充
- **徽章**: 3D效果，发光边框
- **排行榜**: 竞技感设计

---

## 📱 响应式设计 (Responsive Design)

### 移动端优化
- **触控目标**: 最小44px点击区域
- **字体尺寸**: 移动端适当放大
- **间距**: 充足的留白确保舒适操作
- **手势**: 支持滑动、长按等手势

### 桌面端增强
- **悬停效果**: 丰富的鼠标悬停反馈
- **大屏布局**: 网格布局，信息密度优化
- **快捷键**: 键盘导航支持

---

## ♿ 无障碍设计 (Accessibility)

### 颜色对比
- **文本对比度**: 至少4.5:1的对比度
- **状态指示**: 不仅依赖颜色，配合图标
- **色盲友好**: 色彩+形状双重编码

### 动画控制
- **减少动效**: 尊重 `prefers-reduced-motion` 设置
- **可选动画**: 提供动画开关选项

---

## 🎨 设计变量 (Design Tokens)

### 圆角系统
```css
--radius-pill: 9999px      /* 胶囊按钮 */
--radius-mega: 2rem        /* 大卡片 */
--radius-xl: 1.5rem        /* 常规卡片 */
--radius-lg: 1rem          /* 小组件 */
```

### 字体层次
```css
--text-5xl: 3rem           /* 主标题 */
--text-4xl: 2.25rem        /* 页面标题 */
--text-2xl: 1.5rem         /* 组件标题 */
--text-base: 1rem          /* 正文 */
```

### 权重系统
```css
--font-weight-bold: 700    /* 标题 */
--font-weight-semibold: 600 /* 按钮 */
--font-weight-medium: 500   /* 标签 */
--font-weight-normal: 400   /* 正文 */
```

## 📋 代码实践 (Code Practices)

### Tailwind 类命名
- 优先使用设计变量而非固定值
- 组件一致性：相同类型组件使用相同样式
- 响应式优先：mobile-first 设计

### 组件结构
- **容器**: 使用语义化的容器结构
- **装饰**: 分离装饰性元素和功能性元素
- **状态**: 清晰的状态管理和视觉反馈

### 性能优化
- **动画**: 使用 `transform` 和 `opacity` 进行动画
- **图片**: 合理使用 WebP 格式和懒加载
- **代码分割**: 按需加载组件
